# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in this portfolio site, please report it to:

**Email:** gubaupol@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

I will respond as quickly as possible and work with you to address the issue.

## Security Measures

This portfolio implements the following security best practices:

- **HTTPS Only**: All traffic is encrypted via HTTPS
- **Content Security Policy**: Implemented via Vercel headers
- **No User Data Collection**: No personal data is stored or tracked
- **Dependency Updates**: Regular updates of npm packages
- **Static Site**: No server-side vulnerabilities (Astro SSG)
