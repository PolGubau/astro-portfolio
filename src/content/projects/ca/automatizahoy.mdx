## Landing page per a un estudi de disseny i web

Tactil és un estudi creatiu que ofereix **disseny i desenvolupament personalitzats** per a marques ambicioses que busquen experiències digitals **ràpides**, **escalables** i **úniques**.

El seu objectiu és trencar amb el món de les plantilles genèriques perquè **les plantilles són per a presentacions de diapositives, no per al teu lloc web**.

## Visió

Els llocs web moderns solen caure en dos extrems: plantilles sobre-dissenyades però amb un rendiment baix, o temes ràpids però sense ànima visual. At Tactil, tanquem aquesta bretxa oferint interfícies fetes a mà, interaccions subtils i bases de codi amb bon rendiment, tot adaptat a l'ADN de la marca.
<Image src={landing} alt="Landing de Tactil"  height={700} class={"show-animation"} />

Construïm llocs web que:

* Converteixen millor en ser intencionals
* Escalen amb el teu negoci
* Es carreguen extremadament ràpid
* Se senten vius gràcies al moviment i al detall

El nostre eslògan ho resumeix:

> **Disseny i desenvolupament** personalitzats per a marques ambicioses.
> Experiències digitals ràpides🚀, úniques🦄 i escalables📈.

## Per què Astro?

La web no necessita més megabytes de JS només per renderitzar contingut estàtic.

Utilitzem **[Astro](https://astro.build)** com a base per seguir oferint:

* **Zero-JS per defecte**, ideal per a landing pages i SEO
* **Arquitectura d'illes**, permetent interactivitat selectiva
* **Temps de construcció ràpids** i optimitzacions integrades

Combinat amb les **illes de React** i **TailwindCSS**, Astro ens dóna el poder d'oferir un rendiment en temps real *i* interfícies expressives.

| Eina / Llibreria | Propòsit |
| -------------------- | ------------------------------------- |
| `Astro` | Generador de llocs estàtics |
| `React` | Illes d'interfície d'usuari interactives |
| `TailwindCSS` | Estils basats en utilitats |
| `@astrojs/mdx` | Contingut ric amb Markdown i JSX |
| `@astrojs/rss` | Suport RSS per a futurs articles del blog |
| `@astrojs/sitemap` | Generació de sitemap per a SEO |
| `@astrojs/react` | Integració de React per a Astro |
| `@polgubau/animated` | La meva pròpia llibreria de components d'animació |
| `astro-meta-tags` | Generació dinàmica de metaetiquetes |
| `oxlint` | Linter ràpid per a projectes JS/TS moderns |
| `sharp` | Processament i optimització d'imatges |
