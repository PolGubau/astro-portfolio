Aquesta és la meva participació al hackathon JOBarcelona 2023. Beecipes és una aplicació de receptes que utilitza l'API de Spoonacular per obtenir receptes i mostrar-les de manera fàcil per a l'usuari. **Es va fer en menys de 24 hores com a repte**.

- Aquesta aplicació està construïda amb React i Recoil.

- Utilitza l'**API de Spoonacular** per obtenir i mostrar les receptes.

## Funcionalitats

- Cercar receptes per nom
- Cercar receptes aleatòriament
- Emmagatzematge local per desar receptes
- Reducció de crides a l'API per estalviar dades

Aquestes funcionalitats estan dissenyades per **millorar l'experiència de l'usuari proporcionant una funcionalitat útil i reduint la quantitat de dades que s'han d'obtenir de l'API**. La capacitat de cercar receptes per nom permet als usuaris trobar ràpidament la recepta que busquen, mentre que la capacitat de cercar receptes de forma aleatòria ofereix una manera divertida i senzilla de descobrir noves receptes.

L'addició de l'emmagatzematge local també millora l'experiència de l'usuari en permetre als usuaris desar les seves receptes preferides i accedir-hi sense connexió. Això redueix la dependència de la connexió a Internet i permet als usuaris accedir a les seves receptes desades en qualsevol moment.

Finalment, en reduir el nombre de crides a l'API, podem estalviar en l'ús de dades i millorar la velocitat general de l'aplicació. Aquesta és una consideració important per als usuaris que puguin tenir plans de dades limitats o connexions a Internet més lentes.

## **Instal·lació**

Aquest projecte està construït amb Vite.

1. Clona el repositori

   ```bash
   git clone https://github.com/PolGubau/hackathon-JOBarcelona-recipes-2023.git
   ```

2. Instal·la les dependències

   ```bash
   pnpm i
   ```

3. Executa el projecte

   ```bash
   pnpm run dev
   ```

4. Obre el projecte al teu editor de codi preferit.

## **Ús**

Instruccions sobre com utilitzar el projecte.

Per utilitzar l'aplicació, has d'obtenir una clau d'API de [Spoonacular](https://spoonacular.com/food-api).

Després d'obtenir la clau d'API de Spoonacular, pots seguir aquests passos per utilitzar l'aplicació:

Obre el projecte al teu editor de codi preferit.
Ves al fitxer `src/models/url.ts` i substitueix `YOUR_API_KEY` per la teva clau d'API real obtinguda de Spoonacular.
Desa els canvis realitzats al fitxer `src/models/url.ts`.
Obre el projecte utilitzant:

```bash
npm run dev
```

Ara ja hauries de poder utilitzar l'aplicació per cercar receptes per nom o de forma aleatòria i desar-les a l'emmagatzematge local.
Nota: Assegura't de tenir una connexió a Internet estable mentre utilitzes l'aplicació per evitar qualsevol error en obtenir les dades de l'API.
