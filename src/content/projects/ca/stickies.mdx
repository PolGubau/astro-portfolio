# Stickies

## Què és Stickies?

Stickies és una aplicació web per prendre notes i gestionar-les. Pots crear notes privades i classificar-les en categories.

Ha estat un projecte paral·lel durant alguns mesos on vaig aprendre desenvolupament web modern i va ser la clau per convèncer-me de centrar el meu aprenentatge en les **tecnologies reactives**.

## Stack tecnològic

Stickies es basa en tecnologies modernes de desenvolupament web com:

- **React** com a framework principal per al desenvolupament de JavaScript; permet crear components reusables i gestionar l'estat de l'aplicació.

- **Redux** com a gestor d'estat; permet gestionar l'estat de l'aplicació de manera global i poder-hi accedir des de qualsevol component.

- **TypeScript** com a llenguatge principal; permet tenir un codi tipat i poder detectar errors en el codi abans d'executar-lo.

- **Styled Components** com a framework de CSS principal; permet crear components amb els seus propis estils i poder-los reutilitzar a l'aplicació.

- **React Icons** com a llibreria d'icones principal; permet utilitzar icones a l'aplicació d'una manera senzilla.

## My role in Stickies

I have been the only developer of the project, I have been in charge of the design and development of the application.

So far I have developed the following features:

- **Idea and concept** of the application.

- **Design** of the service architecture.

- **Prototyping** of the application in design tools.

- **Development** of the application in React.

- **Deployment** of the application in Vercel.

## What I learned

Stickies has been the mind changing project that has allowed me to aim to be a **Frontend Developer**. I have learned a lot about modern web development and I have been able to apply it in a real project.

I have learned to use **React** as a framework for web development, to manage the state of the application with **Redux**, to use **Typescript** as a programming language, to use **Styled Components** as a CSS framework and to use **React Icons** as an icon library.

## What's next

During 2023, I developed a modern version of Stickies using **Nextjs** and **Tailwind CSS**.
