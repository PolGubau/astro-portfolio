# Stickies

## ¿Qué es Stickies?

Stickies es una aplicación web para tomar notas y gestionarlas. Puedes crear notas privadas y clasificarlas en categorías.

Ha sido un proyecto paralelo durante varios meses en el que aprendí sobre el desarrollo web moderno y fue la clave para convencerme de enfocar mi aprendizaje en las **tecnologías reactivas**.

## Stack tecnológico

Stickies está basado en tecnologías modernas de desarrollo web como:

- **React** como framework principal para JavaScript: permite crear componentes reutilizables y gestionar el estado de la aplicación.

- **Redux** como gestor de estado: permite gestionar el estado de la aplicación de forma global y acceder a él desde cualquier componente.

- **TypeScript** como lenguaje principal: permite tener código tipado y detectar errores antes de ejecutarlo.

- **Styled Components** como framework CSS principal: permite crear componentes con sus propios estilos y reutilizarlos en la aplicación.

- **React Icons** como biblioteca de iconos principal: permite usar iconos en la aplicación de forma sencilla.

## Mi rol en Stickies

He sido el único desarrollador del proyecto, encargándome del diseño y el desarrollo de la aplicación.

Hasta ahora he desarrollado las siguientes funcionalidades:

- **Idea y concepto** de la aplicación.

- **Diseño** de la arquitectura del servicio.

- **Prototipado** de la aplicación en herramientas de diseño.

- **Desarrollo** de la aplicación en React.

- **Despliegue** de la aplicación en Vercel.

## Lo que aprendí

Stickies ha sido el proyecto que cambió mi mentalidad y me permitió aspirar a ser **Desarrollador Frontend**. He aprendido mucho sobre el desarrollo web moderno y he podido aplicarlo en un proyecto real.

He aprendido a usar **React** como framework para el desarrollo web, a gestionar el estado de la aplicación con **Redux**, a usar **TypeScript** como lenguaje de programación, **Styled Components** como framework CSS y **React Icons** como biblioteca de iconos.

## ¿Qué viene después?

En 2023 desarrollé una versión moderna de Stickies usando **Next.js** y **Tailwind CSS**.
