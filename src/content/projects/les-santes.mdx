## Why this project exists

I live in Mataró and I have been enjoying **Les Santes** more than 20 years. 

Each time I found myself (and everyone around me) pulling out the printed programme to check *"what time is La Passada"* or trying to calculate *"where should the Gegants be now if they departed 90 minutes ago"*...

Mataró deserves better tooling. 

So I reached out to the Culture Department, built the app, and published everything as open source.

### Making Les Santes more accessible

Not only does the app provide real-time event info and a live map, but it also makes the festival more accessible for everyone:
- **Offline support** for venues with poor signal
- **Catalan-first** language with English fallback
- **Intuitive UI** for all ages, from teenagers to grandparents
- **Free and open source** for anyone to use or contribute to 


A solo, public-good project. No client, no budget, no team. Just a problem worth solving.

## The app

The mobile app (React Native + Expo) is built around one question: **what's happening right now?**

### Ara (Live events)

Open the app and immediately see what's on: a hero card with the current event and a strip of upcoming ones. No setup, no login.

<Image src={appAra} alt="App Ara view" class="max-h-[60vh] w-auto rounded-2xl my-6" />

### Mapa (Interactive map)

Static events appear as pins on the map. Moving parades (cercaviles, gegants) show their **real-time estimated position** as an animated route polyline, calculated by interpolating elapsed time along the planned route.

<Image src={appMapa} alt="App map with live event routes" data-reveal="bottom" data-easing="soft" class="max-h-[60vh] w-auto rounded-2xl my-6" />

### Agenda (Full programme)

The full schedule, filterable by day, category, event type, or proximity. Plus a favourites system so nothing slips through.

A scrollable archive of every official festival poster since 1892, plus a swipeable lightbox of the Gegants postcards.

## Key design choices

<div class="not-prose grid grid-cols-1 sm:grid-cols-2 gap-3 my-6" data-stagger="80">
  <div data-reveal="bottom" data-easing="soft" class="flex flex-col gap-1.5 rounded-xl border border-neutral-200 bg-neutral-50 p-4">
    <span class="text-sm font-semibold">No login. No accounts.</span>
    <span class="text-sm leading-relaxed text-neutral-500">Festival attendees just want to know what's on. The app is 100% anonymous and it **does not collect any personal data**, open it and go. This also simplified App Store and Play Store approval dramatically.</span>
  </div>
  <div data-reveal="bottom" data-easing="soft" class="flex flex-col gap-1.5 rounded-xl border border-neutral-200 bg-neutral-50 p-4">
    <span class="text-sm font-semibold">Offline-first</span>
    <span class="text-sm leading-relaxed text-neutral-500">Festival venues have terrible signal. All event data is cached locally with a timestamp, so the app keeps working without internet. A banner shows the cache age when offline.</span>
  </div>
  <div data-reveal="bottom" data-easing="soft" class="flex flex-col gap-1.5 rounded-xl border border-neutral-200 bg-neutral-50 p-4">
    <span class="text-sm font-semibold">Catalan as the primary language</span>
    <span class="text-sm leading-relaxed text-neutral-500">Les Santes is a Catalan cultural festival. The app speaks Catalan first, with English & Spanish as fallback. Not an afterthought - it was the first decision made.</span>
  </div>
  <div data-reveal="bottom" data-easing="soft" class="flex flex-col gap-1.5 rounded-xl border border-neutral-200 bg-neutral-50 p-4">
    <span class="text-sm font-semibold">AVIF everywhere</span>
    <span class="text-sm leading-relaxed text-neutral-500">Converting the full poster archive to AVIF brought a ~97% size reduction from 5.2 MB of PNGs to ~146 KB for all app screenshots.</span>
  </div>
  <div data-reveal="bottom" data-easing="soft" class="flex flex-col gap-1.5 rounded-xl border border-neutral-200 bg-neutral-50 p-4 sm:col-span-2">
    <span class="text-sm font-semibold">MapLibre in a WebView instead of native maps</span>
    <span class="text-sm leading-relaxed text-neutral-500">Avoids Google Maps and Apple Maps SDKs entirely - no privacy implications, no native setup overhead. MapLibre GL JS runs in a WebView with custom MapTiler tiles. Full control, no tracking.</span>
  </div>
</div>

## The admin panel

A password-protected back-office embedded in the landing. The festival organiser can create, edit, and cancel events with real-time announcements pushed to all app users.

<Image src={adminHome} alt="Admin panel - event list" data-reveal="bottom" data-easing="soft" class="rounded-2xl my-6" />

Event creation handles both fixed-location events and moving parades (uploaded as GeoJSON routes from geojson.io). Cancellations require a reason that is shown to users inside the app.

<Image src={adminEditEvent} alt="Admin panel - edit event form" data-reveal="bottom" data-easing="soft" class="rounded-2xl my-6" />

## Tech at a glance

| Layer | Stack |
|---|---|
| **Mobile app** | React Native, Expo SDK 54, Expo Router |
| **State & cache** | Zustand, AsyncStorage |
| **Map** | MapLibre GL JS in a WebView, MapTiler tiles |
| **Landing** | Next.js 15, Tailwind CSS v4, shadcn/ui |
| **Backend** | Supabase (PostgreSQL, Auth, Realtime) |
| **i18n** | Catalan primary, English fallback |
| **Images** | AVIF for all media |
